Привет! Спасибо большое за комментарии!
Поправила следующие ключевые моменты:

1) добавила инкрементальную загрузку доставок, в новое поле order_ts теперь пишется дата заказа, перед загрузкой данных из API из таблицы
stg.api_deliveries_raw читается максимальная дата заказа и потом она подсталяется в реквест в параметр from. Таким образом грузятся только новые доставки.
Если таблица пустая, то у нас нет максимальной даты заказы, вместо нее подставляется 2023-01-01 и соотвественно данные грузятся с этой даты.

2) Для курьеров и ресторанов - создала промежуточные таблицы _tmp, из них данные грузятся в таблицы _raw. 
При добавлении новых данных - добавляем новые записи, проставляем first_load_ts. 
Если в запросе пришли те же записи, что уже есть в таблице - ничего не добавляем, но обновляем last_load_ts.
Если в запросе не прошло каких то записей, которые уже есть в таблице - проставляем для них дату в поле delete_ts

3) Данные по доставкам в DDS теперь грузятся инкрементально, только записи по заказам, которых ранее не было в таблице
